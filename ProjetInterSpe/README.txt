# Billard


# Preparer un remote :
git remote add origin https://github.com/mrecuerda/Billard

# Ajouter un nouveau fichier (au commit) :
git add <file>

# Voir les modifications :
git status

# Enregistrer les modifications (local) :
git commit -m <message>

# Envoyer les modifications :
git push -u origin master

# Recevoir les modifications :
git pull
